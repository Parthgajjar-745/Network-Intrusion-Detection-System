UNDER THE HOOD

How NIDS detects intrusions

Two detection rules, applied to every IP and port that comes through the system.

01

Request Logging

Every time an IP sends data, its timestamp is recorded in an in-memory log, keyed by IP address.

02

DDoS Check

If an IP sends more than 5 requests within a 3 second window, it is flagged as a possible DDoS source.

03

Port Tracking

Each unique port an IP touches is added to a set. Repeated ports don't count twice.

04

Port Scan Check

If an IP accesses more than 4 different ports, it is flagged as a possible port scanner.
